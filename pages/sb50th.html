50주년 E역사관
프로젝트 개요

저축은행중앙회 창립 50주년을 기념하는 사이트를 구축했습니다. 기획, 디자인, 개발 전 과정에 참여했으며, 50주년을 강조하기 위해 각 페이지에 애니메이션을 적용하여 디자인의 일관성을 유지했습니다. 또한, PC와 모바일에서 최적의 사용자 경험을 제공하기 위해 적응형 웹 디자인을 적용했습니다.

visit website(↗↗)

50th Anniversary of the Korea Federation of Savings Banks keyvisual

1. 메인 페이지

애니메이션을 활용해 행성이 회전하는 비주얼을 적용, 50주년의 의미를 강조했습니다.

2. 기념사 페이지

스크립트를 사용해 페이지 로드 시 콘텐츠가 서서히 나타나는 효과를 구현했습니다.

3. 엠블럼 페이지

50주년 기념 로고를 '무한대' 형상으로 애니메이션을 통해 표현했습니다.

  • 2. 기념사 페이지

  • 연혁 페이지

    각 년도를 스크롤 시 sticky 기능을 적용해 해당 년도가 화면에 고정되도록 구현했습니다.

  • 숫자로 보는 50년 페이지

    콘텐츠를 효과적으로 보여주기 위해 인터랙션 효과를 추가, 정보를 보다 집중적으로 전달했습니다.

  • 50년의 이야기 페이지

    Swiper를 사용해 각 년도를 대표하는 이미지를 간편하게 탐색할 수 있도록 구현했습니다.

E역사관을 손쉽게 들여다 볼 수 있는
역사와 연혁

Swiper를 활용해 슬라이더 기능을 추가하여, E역사관의 역사와 연혁을 손쉽게 탐색할 수 있도록 하고, 콘텐츠를 돋보이게 구성했습니다.
