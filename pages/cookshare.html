사용자 경험 개선을 위한
인터페이스 개발

Debounce 커스텀 훅을 적용하여 별도의 로직 작성 없이 효율적이고 일관된 방식으로 효율적이고 일관된 방식으로 개발을 진행했습니다.
1. 회원가입 시 사용자 입력을 디바운싱 처리하여 서버 요청을 최소화했습니다.
2. 카카오 지도 API를 활용하여 사용자의 키워드 입력에 따라 반응하는 지도 검색 기능을 구현, 입력 상태가 디바운싱을 통해 최적화되도록 설계하여 사용자의 경험을 개선했습니다.

왼쪽 - 회원가입 페이지 , 오른쪽 - 지도 검색 기능

애플리케이션 상태 관리 및 라우팅 최적화
유지보수 간편하게 개발

react-router-dom을 사용하여 라우팅 로직을 중앙화, 간소화하여 동적인 페이지 구현을 최적화했습니다. 또한, Axios를 사용하여 서버 연동한 데이터를 Redux Toolkit의 createSlice 함수를 활용하연 게시판 기능의 상태 관리를 간소화시켜 유지보수성이 용이하도록 향상시켰습니다.
