사용자 인터페이스 및 인터랙션
모달 및 동적 요소 개발

예약 도메인의 데이터베이스 설계를 담당, Spring Data JPA를 이용하여 CRUD 기능을 구현했습니다.
JSTL과 JavaScript를 활용하여 같은 모달창으로 이미지 상세보기와 예약 창을 구현했습니다. 사용자와의 상호작용을 효과적으로 관리하면서 웹사이트의 기능성을 향상시켰습니다.
옵션을 선택할 때 마다 실시간으로 예약 정보를 확인할 수 있도록 했습니다.

프론트엔드 개발
지도 API 활용, 동적 UI 구성

Kakao 지도 API를 통해 농장 위치 정보를 시각화, 사용자가 쉽게 위치를 확인할 수 있도록 구현했습니다.
AJAX를 통해 서버와 비동기적으로 통신하여 사용자 인터페이스를 동적으로 업데이트했습니다. 이를 통해 페이지 새로고침 없이 실시간으로 데이터를 로드하고 사용자 요청을 신속 반영할 수 있습니다.
