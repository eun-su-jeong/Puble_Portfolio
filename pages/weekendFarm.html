주말농장 예약 플랫폼
"이 플랫폼은 주말농장을 찾고 예약하는 기존의 번거로운 과정 — 주로 전화와 분산된 정보에 의존 — 을 개선하기 위해 개발되었습니다. 사용자가 쉽게 농장을 검색하고 상세 정보 및 다른 이용객의 리뷰를 참고하여 예약할 수 있도록 함으로써, 도시 거주자들이 자연에 보다 편리하게 접근할 수 있도록 도와줍니다."

weekend farm keyvisual

Back-end : 데이터베이스 설계 및 관리와 페이지네이션 로직 구현
예약 도메인의 데이터베이스 설계를 담당, Spring Data JPA를 이용하여 데이터의 일관성과 효율성을 보장하는 CRUD 기능을 구현 데이터 처리의 성능 최적화를 위해 `Pageable` 인터페이스를 활용하여 복잡한 페이지네이션 로직을 개발, 이를 서버 측에서 구현하여 사용자의 데이터 요청 처리 속도를 개선

프론트엔드 개발
지도 API 활용, 동적 UI 구성

Kakao 지도 API를 통해 농장 위치 정보를 시각화, 사용자가 쉽게 위치를 확인할 수 있도록 구현
AJAX를 통해 서버와 비동기적으로 통신, 사용자 인터페이스를 동적으로 업데이트, 이를 통해 페이지 새로고침 없이 실시간으로 데이터를 로드하고 사용자 요청 신속 반응

사용자 인터페이스 및 인터랙션
모달 및 동적 요소 개발, 검색 기능과 실시간 필터링

JSTL과 JavaScript를 활용하여 사용자가 정보를 입력하고 확인할 수 있는 모달 창과 예약 시스템을 구현. 이는 사용자와의 상호작용을 효과적으로 관리하면서 웹사이트의 기능성을 향상.
사용자가 농장을 쉽게 찾을 수 있도록 모든 페이지에 통합된 검색 기능을 개발.`header.jsp` 파일에 포함된 검색창을 통해 일관된 사용자 인터페이스를 제공.
