통합모바일 앱/ODS 구축
프로젝트 개요

프로젝트 개요 사용자에게 편리한 서비스를 제공하는 한국투자저축은행 통합모바일앱 “key뱅크” 와, 사내 운영시스템 ODS 구축 프로젝트 입니다. Websquare5와 Eclipse를 기반으로 개발, 모바일 앱과 ODS를 개별적으로 구축하여 기존의 서비스를 더 편리하게 개선하였습니다.

앱을 설치하여 운영되는 서비스를 볼 수 있습니다

keybank keyvisual

1. 메인페이지

대출 만기일을 한눈에 볼 수 있도록 볼드와 하이라이트 처리를 했으며, badge와 progress bar를 통해 눈에 잘 볼 수 있도록 작업했습니다.

2. 상품 페이지

카드형식으로 상품들을 만들어 한눈에 정보를 보고, 쉽게 원하는 페이지로 이동할 수 있게 구축했습니다.

3. 상품 상세페이지

필요한 내용을 쉽게 볼 수 있도록 아코디언형식으로 만들었습니다.

  • 1. 메인 페이지

  • 2. 상품 페이지

  • 3. 상품 상세페이지

  • 오픈뱅킹 등록 페이지

    카드형식의 체크박스로 만들어 한눈에 정보를 볼 수 있게 구축

  • 한도조회 결과 페이지

    조회된 결과를 카드형식으로 정보를 한눈에 볼 수 있게 구축

  • 대출 상세페이지

    간단한 정보를 함게 볼 수 있도록 라디오버튼에 작성하여 편의성을 제공

개발하기 편리한 프레임워크 사용
컴포넌트화하여 작업

Back-end : 데이터베이스 설계 및 관리와 페이지네이션 로직 구현
예약 도메인의 데이터베이스 설계를 담당, Spring Data JPA를 이용하여 데이터의 일관성과 효율성을 보장하는 CRUD 기능을 구현 데이터 처리의 성능 최적화를 위해 `Pageable` 인터페이스를 활용하여 복잡한 페이지네이션 로직을 개발, 이를 서버 측에서 구현하여 사용자의 데이터 요청 처리 속도를 개선

개발하기 편리한 프레임워크 사용
컴포넌트화하여 작업

Vue3와 Quasar 프레임워크를 활용하여 재사용 가능한 컴포넌트 설계 및 개발을 하여 페이지를 구성했습니다.

image

사용자에게 일관된 서비스를 제공하는
웹표준과 웹접근성 준수 및 테스트

통합 테스트 및 QA 테스트를 수행하여 웹 접근성 인증 마크를 획득하여 웹사이트의 접근성과 법적 준수를 보장하였습니다. 또한, 다양한 디바이스에서 테스트와 디버깅을 실시하여 일관된 서비스 제공으로 사용자 경험을 개선했습니다.
