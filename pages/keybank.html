통합모바일 앱/ODS 구축
프로젝트 개요

한국투자저축은행 사용자에게 더욱 편리한 서비스를 제공하기 위해 통합 모바일 앱 “key뱅크”와 사내 운영 시스템 ODS를 구축하는 프로젝트입니다. 이 프로젝트는 Websquare5와 Eclipse를 기반으로 개발되었으며, 모바일 앱과 ODS를 개별적으로 개발하여 기존 서비스를 크게 개선하였습니다. 이를 통해 사용자 경험과 운영의 효율성을 향상시켰습니다.

앱 설치로 서비스를 이용할 수 있습니다.

keybank keyvisual

1. 메인페이지

대출 만기일을 한눈에 볼 수 있도록 볼드와 하이라이트 처리했으며, badge와 progress bar를 통해 가독성을 높였습니다.

2. 상품 페이지

상품들을 카드 형식으로 구성하여 정보를 쉽게 파악하고, 원하는 페이지로 빠르게 이동할 수 있도록 구축했습니다.

3. 상품 상세페이지

필요한 내용을 쉽게 볼 수 있도록 아코디언 형식으로 구현했습니다.

  • 1. 메인 페이지

  • 2. 상품 페이지

  • 3. 상품 상세페이지

  • 오픈뱅킹 등록 페이지

    체크박스를 카드 형식으로 구성하여 정보를 한눈에 볼 수 있도록 구축했습니다.

  • 한도조회 결과 페이지

    조회된 결과를 카드 형식으로 구성하여 정보를 쉽게 파악할 수 있도록 했습니다.

  • 대출 상세페이지

    간단한 정보를 라디오 버튼과 함께 제공하여 사용자 편의성을 높였습니다.
